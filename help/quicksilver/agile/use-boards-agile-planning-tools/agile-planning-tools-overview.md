# Overzicht van flexibele planningtools: artikelindex

<!-- Audited: 5/2025 -->

Wanneer u een board maakt met behulp van ad-hoc- en verbonden kaarten, kunt u gebruikmaken van extra Agile-planningtools.

De volgende gereedschappen zijn handig voor Kanban:

* WIP-limieten voor kolommen
* Kaarten groeperen in stalen
* Een inlaatkolom die kan worden gebruikt voor een backlog

Tot de nuttige gereedschappen voor Scrum behoren:

* Werkstromen en herhalingsgebieden
* Kaartlijst of -achterstand
* Iteratiemetriek

>[!IMPORTANT]
>
>Workstreams en bijbehorende tools zijn alleen beschikbaar voor een specifieke groep klanten.

Je kunt al deze tools gebruiken op een manier die geschikt is voor jou en je team, zonder dat je je hoeft te houden aan een specifieke Agile-methodologie.
