Een deel van de functionaliteit is vrijgegeven vóór de release van 2019.4. Om te zien wat vóór de versie 2019.4 werd vrijgegeven, zie [ 2019.4 andere verhogingen ](../../../../product-announcements/product-releases/quarterly-release-archive/2019.4-release-activity/2019-4-other-enhancements.md).

Hier volgt de verwijderingstijdlijn voor deze SSO-opties:

* Beperkte steun: augustus 2018 - januari 2019

  Belangrijke kwesties die tijdens deze periode zijn ingediend, worden opgelost; problemen met een lage ernst worden niet opgelost.

* Afschrijving: januari 2019 - november 2019 (met de release van 2019.4)

  Niet ondersteund.

* Niet meer beschikbaar in Workfront voor nieuwe configuraties: november 2019
* Vereiste migratie voor bestaande klanten: begin 2020

### Ondersteuning voor TLS 1.0 en 1.1 loopt af {#tls-1-0-and-1-1-support-is-ending}

Om een optimale beveiliging te bieden, eist Workfront dat alle integratie en het webverkeer die TLS 1.1 of eerder gebruiken, worden geüpgraded naar TLS 1.2. In de voorvertoningsomgeving is TLS 1.0 al uitgeschakeld.
