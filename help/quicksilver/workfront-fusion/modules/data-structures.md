# Gegevensstructuren in [!DNL Adobe Workfront Fusion]

Een gegevensstructuur is een document waarin de indeling van de gegevens waarnaar wordt overgebracht, in detail wordt beschreven [!DNL Adobe Workfront Fusion]. Gebaseerd op dit document, kan de scenario redacteur erachter komen welke module terugkeert of ontvangt welk soort gegevens. De documenten van de gegevensstructuur worden het meest meestal gebruikt voor het in series vervaardigen/ontleden van gegevensformaten zoals JSON, XML, CSV en anderen.

U kunt een gegevensstructuur maken door op de knop [!UICONTROL Create a new data structure] in de [!UICONTROL Data structure overview] of in de instellingen van de module waarvoor de gegevensstructuurspecificatie vereist is.

Ondersteunde gegevenstypen worden beschreven in het dialoogvenster [[!UICONTROL Types of modules]](../../workfront-fusion/modules/module-types.md) artikel.

## Gegevensstructuurgenerator

Gegevensstructuren hoeven niet altijd gemaakt te worden. U kunt het gemakkelijker maken door een malplaatje van onze ingebouwde generator te gebruiken. Door een gegevenssteekproef te verstrekken, zal de generator automatisch tot een gegevensstructuur leiden die op de gegevenssteekproef wordt gebaseerd u inging. De gemaakte gegevensstructuur kan vervolgens handmatig worden gewijzigd.
